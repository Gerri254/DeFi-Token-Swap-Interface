# DeFi Token Swap Interface

React.js application for swapping ERC-20 tokens with MetaMask integration.

## Features
- Connect MetaMask wallet
- Select tokens for swapping
- Real-time price calculation
- Transaction execution and tracking
- Responsive design

## Tech Stack
- React.js with TypeScript
- Ethers.js for blockchain interaction
- Tailwind CSS for styling
- MetaMask for wallet connectivity

## Getting Started
1. Clone repository
2. `npm install`
3. `npm start`
4. Connect MetaMask wallet
5. Select tokens and swap!
